# PNL CONSOLIDE (MASTER BASE)

Ce dossier sert a centraliser les resumes PnL des runs valides.

Convention recommandee:

- Un fichier resume par run valide.
- Nom exemple:
  - `pnl_resume_YYYY-MM-DD_HHMM_validated_30m.txt`
  - `pnl_resume_YYYY-MM-DD_HHMM_validated_6h30.txt`

Objectif: ne plus disperser les chiffres entre dossiers.
